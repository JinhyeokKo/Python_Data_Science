Pivot_Tables¶

입력값으로 간단한 열 단위의 데이터를 취하고 그 데이터에 대한 다차원 요약을 제공하는 2차원 테이블로 항목을 그룹핑
GroupBy 집계의 다차원 버전
분할-적용-결합 작업을 하면 분할과 결합 작업이 1차원 인덱스에서 발생하는 것이 아닌 2차원 그리드에서 발생

Additional pivot table options¶

pivot_table signature

pandas.pivot_table(
    data,
    values=None,
    index=None,
    columns=None,
    aggfunc='mean',
    fill_value=None,
    margins=False,
    dropna=True,
    margins_name='All',
    observed=True
)

추가 데이터 검색 Further data exploration¶

In [13]:
# 이상치 제거 --> 시그마 클리핑 연산
quartiles = np.percentile(births['births'], [25, 50, 75])
mu = quartiles[1]
# 표본 평균의 추정치, 0.74는 가우스 분포의 사분위 수에서 비롯
sig = 0.74 * (quartiles[2] - quartiles[0])
In [14]:
births = births.query('(births > @mu - 5 * @sig) & (births < @mu + 5 * @sig)')
In [15]:
# 'day'열을 정수형으로 설정; 원래는 널 값 때문에 문자열
births['day'] = births['day'].astype(int)

# 년, 월, 일로부터 날짜 인덱스 생성
births.index = pd.to_datetime(10000 * births.year +
                              100 * births.month +
                              births.day, format='%Y%m%d')

births['dayofweek'] = births.index.dayofweek
